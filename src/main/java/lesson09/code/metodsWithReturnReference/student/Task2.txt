Напишите приложение, которое позволяет создать трех студентов.

Класс Student со следующими параметрами типа String:
- имя
- фамилия
- id номер
- группа

Запросите у пользователя и создайте трех студентов и выведете информацию о них.

Реализуйте решение двумя способами:
- все действия в методе main
- с помощью отдельного сервисного класса и метода для создания экземпляра класса Student
