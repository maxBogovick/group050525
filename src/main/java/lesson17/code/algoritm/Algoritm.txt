Сложность алгоритмов - Big-O (большая О)
О-нотация

Линейная сложность - O(N), где N- подразумевается размер данных

Постоянная сложность - O(1) - постоянная сложность

Логарифмическая - O(log(N)) - бинарный поиск

- N * log(N) - определенные алгоритмы сортировки
- N * N, или (N^2) - двойные циклы (от 0 до N)
- N * N * N, или (N^3) - тройные циклы

- Exp(N)
- N!
