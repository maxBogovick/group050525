Написать приложение, которое демонстрирует игру "Лотерея"

1) Для игрока выдается несколько лотерейных билетов (например 3), каждый из которых содержит 5 чисел
2) Игроков может быть несколько
3) производится розыгрыш лотереи
4) Выпадают 5 случайных значений, которые не должны повторяться
5) диапазон значений от 1 до 36

Подведение результатов

-------------------------------------------------------------------

Что делаем:

- лотерейный билет -> массив для хранения 5 чисел, которые выбрал Игрок (или случайным образом - билет с заранее заполненными случайными числами)
- игрок - массив для хранения нескольких билетов
- массив для хранения ВСЕХ возможных номеров из которых мы случайным образом получим значения для выигрыша
- массив для хранения выигрышных номеров


Player
LotteryTicket
LotteryResult
PlayLottery
Lottery

LotteryTicket:
- int[] из 5 элементов
- номер билета

- нам необходимо в момент создания нового билета автоматически его заполнить (в нашем случае - случайными неповторяющимися числами)
